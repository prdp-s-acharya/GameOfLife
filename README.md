# Conway's Game of Life

Implementation of Conway's Game of Life in c#

[web demo](https://prdp-s-acharya.github.io/GameOfLife/GameOfLife/)

## Quick Start

### Build project
```console
$ dotnet build
```

### Run
```console
$ dotnet run --no-build
```
or
```console
$ dotnet run --no-build 20 20 0.5
```

where arguments: 
1. no. of rows,
2. no of columns,
3. % of live cells in 1st itteration
