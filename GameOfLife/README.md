# Conway's Game of Life

Implementation of Conway's Game of Life in c#

## Quick Start

### Build project
```console
$ dotnet build
```

### Run
```console
$ dotnet run
```

or
```console
$ dotnet run 20 20 0.5
```

where arguments: 
1. no. of rows,
2. no of columns,
3. % of live cells in 1st itteration
